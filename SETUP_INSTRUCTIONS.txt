NEXT STEPS - BACKEND INTEGRATION:
---------------------------------
The frontend is complete and ready to use! To enable full functionality, you need to:

1. SUPERWALL SETUP:
   - Sign up at https://superwall.com
   - Get your API key from the Superwall dashboard
   - Replace "YOUR_SUPERWALL_API_KEY" in app/_layout.tsx with your actual key
   - Configure your paywall in the Superwall dashboard with placement ID: "pro_upgrade"

2. BACKEND FEATURES NEEDED:
   The following TODO comments mark where backend integration is needed:
   
   - AI Affirmation Generation (OpenAI GPT-5.2)
     Location: app/(tabs)/(home)/index.tsx - generateAffirmation()
     Purpose: Generate personalized affirmations using AI
   
   - Photo Upload to Storage
     Location: app/(tabs)/(home)/index.tsx - pickImage()
     Purpose: Upload journal photos to cloud storage
   
   - Habits Sync
     Location: app/(tabs)/(home)/index.tsx - loadData() and toggleHabit()
     Purpose: Sync habits across devices and track completion history
   
   - Progress Data Sync
     Location: app/(tabs)/profile.tsx - loadProgressData()
     Purpose: Track streaks, badges, and calendar data
   
   - Affirmations Database
     Location: utils/affirmations.ts - loadAffirmationsOffline()
     Purpose: Load top 500 affirmations from backend on first launch
